Данное приложение позволяет работать с базой компаний, их водителями и их транспортными средствами.
Используемая база данных - PostgreSQL, строка подключения находится в файле appsettings.json.
Аутентификация пользователей реализована с помощью с JWT.
Логирование реализовано с помощью Serilog.
Для маппинга используется AutoMapper.
